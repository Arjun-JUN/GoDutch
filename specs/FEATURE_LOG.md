# GoDutch — Feature Log

> This file is the source of truth for all features being planned, in progress, or shipped.
> Each feature links to its spec document in the `features/` folder.

## How to use
- Add a new row when a feature is proposed
- Create a spec file in `features/` and link it here
- Move rows between sections as the feature progresses through the pipeline

## Status legend
| Status | Meaning |
|---|---|
| 💡 Idea | Proposed, not yet specced |
| 📝 Speccing | Spec in progress |
| ✅ Specced | Spec complete, ready for build |
| 🚧 In Progress | Currently being built |
| 🚀 Shipped | Live in the app |
| ❌ Dropped | Decided not to build |
