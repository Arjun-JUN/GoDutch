## How to Use This Log

- **Adding a new brief**: Create a file named `YYYY-MM-DD_topic.md` and add a row to the index above.
- **Updating a brief**: Create a new dated file rather than overwriting the old one. Mark the old row as "Superseded" and link to the new file.
- **Adding weekly monitors**: Save to `weekly-updates/competitor-monitor-YYYY-MM-DD.md` and add a row to the Weekly Monitor Log above.
- **Flagging a review**: If a major competitive event happens (pricing change, new entrant, funding round), add a note to the relevant row and update the review date.
