# Data Visualization

> **Status: Not implemented yet.**

Charts and graphs for GoDutch spending insights.

## Overview

Data visualization is a planned section for the Analytics / Insights screen. Currently GoDutch displays numbers (balances, totals) but has no chart components.

| Chart type | Use case |
|------------|---------|
| Bar chart | Monthly spend by category |
| Line chart | Balance over time |
| Donut / pie | Expense breakdown by person |
| Stacked bar | Group contributions |
